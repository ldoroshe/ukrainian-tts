# ukrainian-tts agent context (public fork safe)

This file documents agent guidance for public forks and intentionally avoids exposing local absolute paths.

## Shared/global rules
- Read `./GLOBAL_AGENTS.md` first (in private workspaces this is typically a symlink to the workspace global rules).
- If `GLOBAL_AGENTS.md` is unavailable in a public fork, follow project-local instructions and contact the repository owner for the canonical global rules.

## Obsidian memory for contributors
- This project uses a shared, private Obsidian vault for agent memory. Public forks do not have access to the private vault. Refer to repository CONTRIBUTING.md or contact the maintainer for guidance.
